# 🛍️ ShopSmart — AI-Powered E-Commerce Chatbot

An interactive and modern e-commerce web application featuring an AI-powered chatbot that assists users with product queries, recommendations, and customer support in real-time.

---

## 🚀 Features

### 🏪 E-Commerce UI
- Responsive landing page with modern design
- Sticky navigation bar with search and menu
- Featured products section with ratings & pricing
- Promotional banner and testimonials
- Clean footer with links and social icons

### 🤖 AI Chatbot
- Real-time AI-based shopping assistant
- Smart product recommendations
- Predefined quick suggestion buttons
- Typing indicator for better UX
- Message timestamps and auto-scroll
- Chat history and clear chat option
